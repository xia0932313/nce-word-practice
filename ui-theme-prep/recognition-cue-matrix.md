# Recognition Cue Matrix

目标不是简单换色，而是把“原作让人一眼识别的体验结构”拆成可用的视觉语法，再转译成原创素材。这样孩子能感到熟悉和兴奋，同时不会直接复刻官方角色、标志和道具。

## 执行边界

可以做：

- 强题材感：开放世界冒险、电影史诗远征、蓝白未来伙伴。
- 强 UI 结构感：装备栏、任务日志、地图卷轴、道具抽屉、奖励槽。
- 强材质感：古代石板、木牌、羊皮纸、锻造金属、软胶按钮、贴纸。
- 强反馈感：符文亮起、火种点亮、星章弹出。

不能做：

- 直接使用原作人物、怪物、武器、道具、标志、专有图案。
- 做到足以被误认为官方截图或官方素材。
- 使用原作名称、原作图案组合或独特轮廓。

## 1. 荒野遗迹冒险

| 需要传达的熟悉感 | 不使用的直接元素 | 原创转译 | 放到 UI 的位置 |
| --- | --- | --- | --- |
| 开放世界探索 | 原作主角、王国种族、官方纹章 | 披风学徒、旅行靴、布带、手作滑翔符 | 顶栏头像、奖励卡、空状态 |
| 古代科技 | 眼形标志、专有符文、官方平板 | 青绿色发光线、几何石板、抽象菱形符文 | 题目面板、进度条、选项边框 |
| 装备收集 | 大师剑、三角徽章、官方盾 | 风化短剑、叶纹盾、青铜扣、试炼星石 | 答对奖励、任务卡图标 |
| 营地怪物 | 原作怪物轮廓、面具比例 | 木面具小怪、石壳守卫、夜光软泥 | 错词、弱项、错误反馈 |
| 荒野地貌 | 官方地图地名、建筑轮廓 | 草坡、远山、浮石、残破石门 | 页面背景、统计页背景 |

识别增强手段：

- 每个主要面板都带一点“古代试炼装置”的边框，不只按钮换色。
- 单词选项做成“装备格”，四个选项有四种不同角饰。
- 正确反馈不是绿色框，而是“符文亮起”；错误反馈不是红框，而是“裂纹警示”。

## 2. 远征史诗奇幻

| 需要传达的熟悉感 | 不使用的直接元素 | 原创转译 | 放到 UI 的位置 |
| --- | --- | --- | --- |
| 旅队远征 | 电影角色脸、服装、队徽 | 斗篷学习者、旅队徽章、地图路线 | 顶栏、任务卡、统计页 |
| 古地图 | 官方地名、地图形状 | 羊皮纸纹理、山脊线、路线点、方位针 | 背景、筛选栏、错词列表 |
| 暗影敌人 | 魔眼、戒灵、专有盔甲 | 暗影面具兵、黑雾骑士、碎石哨兵 | 错词强化、错误反馈 |
| 武器道具 | 魔戒、专属剑、白树 | 叶形短刃、旅人法杖、锻造锤、远征罗盘 | 奖励、选项角标 |
| 电影感 | 直接复刻镜头构图 | 火光、月蓝、铁灰、深绿、边缘暗纹 | 页面整体、按钮层级 |

识别增强手段：

- 每个页面像一页远征日志：顶部是旅队标题，筛选像地图章，统计像路程记录。
- 单词选项做成“地图卡牌”，四个选项分别有蜡封、铁角、路线针、火种角标。
- 正确反馈是“火种点亮”，错误反馈是“暗影侵蚀”。

## 3. 口袋未来伙伴

| 需要传达的熟悉感 | 不使用的直接元素 | 原创转译 | 放到 UI 的位置 |
| --- | --- | --- | --- |
| 蓝白未来伙伴 | 原作机器猫脸、铃铛、口袋比例 | 圆形学习机器人、天线帽、星章、软手套 | 顶栏、奖励、空状态 |
| 道具感 | 任意门、竹蜻蜓、时光机等专有道具 | 记忆灯、发音铃、词汇胶囊、星门学习卡 | 任务卡、按钮、奖励槽 |
| 儿童动画 | 原作角色比例和线条 | 圆角、贴纸、抽屉、软胶按钮、明亮桌面 | 全部控件 |
| 问题怪物 | 原作角色和反派 | 作业云、错词小影、乱序方块、瞌睡泡泡 | 错词本、弱项提醒 |
| 家庭学习 | 原作房间布局 | 未来课堂、桌面、柔和网格、窗光 | 页面背景 |

识别增强手段：

- 页面像“学习伙伴控制台”，控件像道具抽屉和胶囊按钮。
- 单词选项像四个道具按钮，每个选项有不同贴纸角标。
- 正确反馈是“星章弹出”，错误反馈是“修理提示”，避免惩罚感。
